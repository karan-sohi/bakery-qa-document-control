# Bakery QA Document Control

Planning prototype for an internal bakery QA document management website.

## What This Includes

- QA admin document library interface
- Custom department folders
- Current document preview in the website
- Fullscreen document modal
- Version history and archive concept
- Review reminder concept
- Regular user read-only portal
- Print, download, and dark mode controls
- Build plan and suggested data model

## Open Locally

Open `index.html` in a browser, or serve the folder locally:

```sh
python3 -m http.server 4173
```

Then visit:

```text
http://127.0.0.1:4173/
```

## Notes

This is a frontend planning prototype. It does not include authentication, backend storage, database tables, real PDF uploads, or production document permissions yet.
